# 1. 哪些有编译错误
第一个
```java
public class Test {  
    static int b;  
    public static void main(String args[]) {  
        int a = 10;  
        int c;  
        if (a > 50) {  
            b = 9;  
        }  
        c = b + a;  
    }  
}
```

第二个：
```java
public class Test {  
     int b;  
    public static void main(String args[]) {  
        Test test =new Test();  
        int a = 10;  
        int c;  
        if (a > 50) {  
            test.b = 9;  
        }  
        c = test.b + a;  
    }  
}
```
