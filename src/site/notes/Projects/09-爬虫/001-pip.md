- 安装指定包
```bash
pip install ipython
```

- 显示已经安装的包
```bash
pip list
# 更全：
pip freeze
```

- 卸载
```bash
pip uninstall ipython
```

- 修改下载源
```bash
pip install ipython -i https://pypi.douban.com/simple
```
