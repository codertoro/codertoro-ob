- `Boolean` 只有两个值，true 和false
- 其他类型转换为`Boolean` 的转换规则：
- 假值：
	- false
	- +0  -0
	- NaN
	- ""     空字符串
	- null
	- undefined
- 真值
	- true
	- 非0数值
	- 非空字符串
	- **`[]空数组`**   和 **`[]空对象`** 
	- **function 函数**
- 构造函数就是`Boolean()` 函数，常用于强制类型转换
