# 1. 对象分类
- 内置对象
	- `String`  
	- `Array`
	- `Date`
	- `Math`
- 自定义对象
# 2. `String` 对象的常用方法
- `str.length` 获取长度
- `str.toLowerCase()` 大写转小写
- `str.toUpperCase()` 小写转大写
- `str.charAt(n)` 获取第n+1个字符，n是下标，第一个字符下标是0
- `str.substring(start,end)`  截取字符串的某一部分，截取范围[start , end)
