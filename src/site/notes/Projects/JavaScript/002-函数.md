# 1. 函数的调用
- 直接调用
- 在表达式中调用
- 在超链接中调用
```html
<!DOCTYPE html>
<html>
<head>
  <title>调用 JS 函数示例</title>
  <script>
    function sayHello() {
      alert("你好，世界！");
    }
  </script>
</head>
<body>
  <a href="javascript:sayHello()">点击这里调用函数</a>
</body>
</html>
```
